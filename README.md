# kakabot

### Requirements
- python 3
- [python-telegram-bot](https://github.com/python-telegram-bot/python-telegram-bot)


### Steps
Install the library.
```
pip3 install python-telegram-bot
```

Run the program.
```
python3 main.py
```

---------------------------------------

###Available commands

<ul>
<li>Start: returns welcome message.</li>
<li>Koala: returns random koala picture.</li>
</ul>
